The Bar Editor

On this screen you can view a chosen bar; place and re-arrange elements on it. Disabled Elements: This is the list of the elements which have NOT been chosen to be on the bar.
Enabled Elements: This is the list of the elements which HAVE been chosen to be displayed on the bar. Note that the elements will appear in the horizontal (left-to-right) order in which they appear in this list.
Getting Started: Choose the Basic Bar element in order to get started.
Attributing New Elements: in order to add a new element to the bar, drag it from the Disabled column to the Enabled column. Double click an element in order to change its location and othe settings.
Saving your Bar: in order to SAVE your Bar or changes made to it, click SAVE at th bottom of the screen.
